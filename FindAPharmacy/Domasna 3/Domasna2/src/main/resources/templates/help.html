Help



  • Кога корисникот ќе ја отвори апликација прикажана е почетната страница на апликацијата. На секоја од страниците, во горниот дел може да се забележи навигациско мени. На него се наоѓаат копчињата “Почетна”, “Аптеки”, “Локатор на аптеки”, поле за пребарување - “Search”, “Најави се”, “Регистрирај се” и “Помош”.

  • Со клик на копчето “Почетна” корисникот се враќа на почетната страница, независно од тоа на која страница претходно се наоѓал.

  • Со клик на копчето “Аптеки” на корисникот му се прикажува страница каде се прикажани сите аптеки во рамки на државата и соодветно за секоја од нив можат да се видат информации за името, градот и конкретната адреса. Кога корисникот ќе одбере една од аптеките, на прозорецот се прикажуваат дополнителни информации за самата аптека како работно време, телефонски број и оддалеченост на корисникот од избраната аптека.

  • Со клик на копчето “Локатор” се прикажува мапа на Македонија каде се означени аптеките. Соодветно корисникот има можност за пребарување на аптеки според град така што кога корисникот пребарува според одреден град, на мапата се прикажуваат само аптеките кои што се наоѓаат во тој град.

  • Со полето “Search” овозможено е пребарувањето на соодветна аптека и лоцирање на истата на самата мапа, и соодветно пребарувањето на аптеки според одреден град.

  • Со копчето “Регистрирај се” на корисникот му се прикажува страница каде што може да креира свој профил со тоа што ќе ја пополни формата со свои лични податоци, кои пак информации подоцна може да ги види со кликнување на неговото корисничко име во горниот десен агол.

  • Со копчето “Најави се” на корисникот му се прикажува страница каде што може да се најави на неговиот кориснички профил. Тука потребно е да искористат дел од податоците внесени при креирање на самиот профил.


  • When the user launches the web-based application, the Home page of the application is immediately activated, which in the upper part, in the header section contains a menu, buttons "Home", "Pharmacies", "Locator", field for "Search", "Log in”,“ Sign in”,“Help ”and language support.

  • By clicking on the button "Pharmacies" the user is redirected to a page where all pharmacies are displayed and according to each one, can be seen information such as name, location and address, and by clicking the button “Show more”, we have more information about the pharmacy (such as working hours, some activities, telephone numbers, etc.) and displays the map with a locator for where exactly it is located, as well as the distance to it from the location of the user. From this page, when the "Home" button is clicked, the user is redirected to the home page of the application he was on when he started the application.

  • Clicking on the "Locator" button displays a global map where all pharmacies are initially located. It is possible to search by city, pharmacy name and address, but also by some of the additional information about the pharmacies. When the user searches by a specific city or by name of a pharmacy, the map shows only the pharmacies that are located in that city or if it is by the name of the pharmacy, the map shows all the pharmacies that are under that name in all cities, and their distance from the user's location. If the "Home" button is clicked on this page, the user is redirected to the home page of the application he was on when he started the application.

  • With the field "Search" you can search for a suitable pharmacy, which will be located directly on the map.

  • With the button "Register" the user will be redirected to a new page where he will be able to create his profile with relevant information about him, which information can later be seen by clicking on his username in the upper right corner.

  • With the button "Login" the user is redirected to a new page where he can log in with his previous registration. It is not necessary to enter all the information that was entered when creating the profile, but only the username (e-mail) and its password.

  • In the upper right corner, after the button "Register", the user has the opportunity to change the language from Macedonian to English and vice versa from English to Macedonian.

  • In the lowest part, the footer part of the application, the user can find appropriate contact information.
